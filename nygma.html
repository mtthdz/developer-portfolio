Nygma

  • bootcamp project
  • React fundamentals
  • Firebase

The Nygma Machine is a group React & Firebase project completed during Juno's Web Development Immersive bootcamp. An homage to the 1995 film Batman Forever, our group was assigned the project in an "agency" setting, where we were given little-to-no instructor support to complete.

The project highlights our collective learning in and out of bootcamp, including an implementation of a recursive algorithm, Canvas to create an interactive maze, and Firebase to house player data.

This project was completed alongside Brittany Giles, Cecile Zhang, and Jay Hinds.
