Willow

  • bootcamp project
  • PDF Conversion
  • HTML & SCSS fundamentals

Willow is a multi-page PSD conversion project completed during Juno's Web Development Immersive bootcamp. With only a PSD file and a folder of images, I was tasked to recreate the PSD in HTML with pixel-perfect precision. This project namely showcases the benefits of using SASS/SCSS in a multi-page instance, utilizing SCSS mixins, variables, and functions.
