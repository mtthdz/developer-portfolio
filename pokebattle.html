Pokebattle

  • bootcamp project
  • HTML & SCSS fundamentals
  • jQuery
  • RESTful API data

Pokebattle is a jQuery/API project assigned in Juno's Web Development Immersive bootcamp, which highlights the ability to work in a paired-programming environment. The project is a simple rock paper scissors game with a Pokemon theme, completed with bootcamp classmate Audra Turuta. We utilized Giphy to populate a randomized Pokemon-themed reaction GIF, that plays alongside the game result.
